# ğŸ“ **LAYOUT GUIDELINES**

## Banner Dimensions
- **Large Vinyl Banner**: 6ft wide x 3ft tall (72" x 36")
- **Standard Poster**: 24" x 36" (vertical)
- **Table Display**: 36" x 24" (horizontal)

## Typography
- **Main Title**: 120pt-180pt (bold, sans-serif like Roboto or Montserrat)
- **Tagline**: 48pt-72pt (medium weight)
- **Section Headers**: 36pt-48pt (bold)
- **Body Text**: 24pt-36pt (regular)
- **Fine Print**: 18pt-24pt (light)

## Spacing
- **Margins**: 2-3 inches on all sides
- **Section Spacing**: 1-1.5 inches between major sections
- **Line Height**: 1.4-1.6 for readability

## Visual Elements
- Use **icons** (brain, lock, chart, heart) at 80-120pt size
- Include **subtle gradient backgrounds**
- Add **thin decorative lines** (2-4pt) to separate sections
- Use **boxes/cards** for grouping related info
